ロト7 相棒ジェネレータ（スマホ向け・静的Webアプリ）

このZIPの中身をそのままGitHub Pages等に置くと、スマホのブラウザで動きます。
サーバー側の処理は一切なく、データは端末（ブラウザのローカルストレージ）に保存されます。

特徴
- 当選データ＋セット球（第1回〜第666回）を初期データとして内蔵
- 新しい結果を追加するほど「セット球予測」「セット別の数字ウェイト」が更新される（学習型）
- 毎週1口買いを想定して、1口分だけを“それっぽく”チューニングして提示

使い方（毎週の更新）
1) 画面下の「追加」へ
2) 開催回 / 日付 / セット球 / 本数字7個 / B数字2個 を入力して保存
   - 同じ形式の行を貼り付けて「貼り付け反映」すると、開催回・セット球・数字を自動抽出できます
3) 「ホーム」に戻って「再生成」で今週の1口を更新

バックアップ
- 「設定」→「JSONを書き出し」
- 新端末では「JSONを読み込み」で復元できます
